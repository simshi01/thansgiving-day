# День Благодарения - Интерактивная страница

Интерактивная страница для отображения сообщений благодарности с анимациями в стиле iPhone.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Структура проекта

- `components/MessageBubble.tsx` - Компонент отдельного сообщения с анимацией
- `components/MessageContainer.tsx` - Контейнер для управления пулом сообщений
- `app/page.tsx` - Главная страница с тестовыми данными

## Тестирование анимаций

Страница уже содержит тестовые сообщения для проверки анимаций. Сообщения будут появляться в случайных местах экрана, показываться определенное время (зависит от длины текста) и плавно исчезать.
